Pengertian

Hardware komputer adalah semua komponen fisik pada komputer yang bisa dilihat, disentuh, dan dirasakan secara langsung, serta berfungsi untuk menjalankan sistem komputer secara keseluruhan.

Contoh hardware komputer:

- Perangkat input (alat masukan)

- Perangkat output (alat keluaran)

- Perangkat proses

- Perangkat penyimpanan

- Perangkat pendukung

Perangkat Input

Keyboard

Keyboard adalah perangkat input pada komputer yang digunakan untuk memasukkan data dan perintah dengan cara menekan tombol-tombol tertentu.

Tombol pada keyboard biasanya berisi:

- Huruf (A–Z) → untuk mengetik teks

- Angka (0–9) → untuk input numerik

- Tombol fungsi (F1–F12) → untuk perintah khusus

- Tombol kontrol seperti Ctrl, Alt, Shift → untuk kombinasi perintah

- Tombol navigasi seperti Enter, Backspace, Arrow keys.

Mouse

Mouse adalah perangkat input komputer yang digunakan untuk menggerakkan kursor di layar serta memilih, membuka, dan menjalankan perintah dengan cara klik, geser, atau gulir.

Mouse biasanya memiliki:

- Klik kiri → memilih atau membuka objek

- Klik kanan → menampilkan menu opsi

- Scroll wheel → menggulir halaman ke atas atau ke bawah.
